# Tarea de Minería de Datos y Modelización Predictiva

## Descripción del Proyecto
Este repositorio contiene la tarea práctica de evaluación del módulo de Minería de Datos del máster que estoy cursando. El proyecto se centra en la depuración de datos, la justificación analítica en la toma de decisiones sobre las observaciones, y la implementación de modelos de regresión y clasificación.

El análisis abarca desde la limpieza profunda de un dataset agrícola hasta la interpretación estadística de los resultados, asegurando que las transformaciones y la elección de algoritmos estén correctamente justificadas en base a las observaciones subyacentes.
